Complete

Class Project

This project was the fourth and final project in the sequence for EECS 484: Database Management Systems. For this project we had to implement the partition and probing steps for the popular database management algorithm: Grace Hash Join. The implementation mainly relied on the use of given helper functions as well as a given Bucket method. The basic idea for the partition phase is:

  • Loop through all left pages
  • Loop through all left records
  • Hash the key of the record and place the record in an in memory hash table
  • Flush the B-1 hash table pages to disk for use in the Probing phase
  • Repeat for the right input
